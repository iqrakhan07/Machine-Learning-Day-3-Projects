# Machine Learning Day 3 - Logistic Regression Projects

## 📌 Overview
This repository contains two Machine Learning classification projects built using **Logistic Regression** under **Supervised Learning**.

## 🩺 Project 1: Diabetes Prediction

### Objective
Predict whether a patient has diabetes based on medical information.

### Features
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

### Algorithm
- Logistic Regression

 ### Workflow

- Load the diabetes dataset
- Split data into training and testing sets
- Train a Logistic Regression model
- Evaluate model accuracy
- Predict diabetes status for a new patient
- Display the prediction as **Diabetic** or **Not Diabetic**

### Evaluation
- Accuracy Score
